## Editorial Privacy Rule

Case studies must not publish client or project names unless explicit approval is given for that specific reference. Use anonymized descriptions such as "one of Modeso's enterprise projects," "a Saudi fintech engagement," or "a multi-product software house."

Apply this rule to titles, article text, descriptions, tags, image text, alt text, diagrams, code examples, URLs, and metadata. Architecture details must also be sanitized to exclude internal identifiers, credentials, domains, addresses, and repository names.
